The Image Editor allows users to upload a list of image URLs and automatically apply edits to fit Stacker's image format.

To edit a list of images URLs:

  • 1. Click "Choose FIle" and select a CSV with list of images in the first column. The first cell can be a header (it gets skipped). Here's a link to an example image CSV.
  • 2. Click "Submit"
  • 2. The Image Editor will take each image URL and produce two images ranging from 800x570 to 4000x2857:
  • (a) one scaled to fit the full image in the frame, adding blurred padding where necessary
  • (b) one scaled to fill the entire frame with the image, cropping the picture where necessary
  • 3. The Image Editor defaults to producing a list of the URLs in format A (blurred padding added). To select another format for an image, click the image preview (the selected image will highlight in blue).
  • 4. When you're satisfied with the images, copy and paste the list of image URLs at the top of the results page

    • * Any missing images or image URLs not in the correct format will produce an "Error" message for that image.
